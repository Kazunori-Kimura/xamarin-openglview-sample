# OpenGLViewのサンプル

https://developer.xamarin.com/api/type/Xamarin.Forms.OpenGLView/

↑これをそのまんま実装したプロジェクト。

## はまりどころ

OpenTKはNuGetから持ってくるのではなく、Xamarinが内包している奴(?)を使う。

* `ソリューション エクスプローラー` にて `OpenGLDemo.Android` の `参照` を右クリックし、`参照の追加` を選択
* `アセンブリ` から `OpenTK` のバージョン `0.9.9.3` と `1.0.0.0` を選択
* このとき、大量に `OpenTK` の項目が見えている。名前をマウスオーバーすると該当DLLの格納フォルダが表示されるので、そのフォルダ名で必要な項目を判定する。
* おそらく、参照マネージャーのターゲットに表示されている内容とフォルダ名のバージョン表記が一致しているものを選択すれば問題ないと思われる。
